# dayLife

## Concepts
  - 일일, 주간, 월간, 연간 계획과 목표들을 한 페이지에서 볼 수 있습니다.
  - 단기적인 계획들을 관리하면서 장기적인 목표도 잊지 않도록 상키시켜주고, 궁극적으로 모두 성취할 수 있도록 도와줍니다.
  - 달력 기능으로 편리하게 일정을 관리할 수 있습니다.
